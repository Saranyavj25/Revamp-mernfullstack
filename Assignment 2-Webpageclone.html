Basic Form Registration






HTML ENTITIES

1. Use © to display the copyright symbol:

Example: © 2024 My Company


2. Use ® for the registered symbol:

Example: tcs® is a leading IT company.


3. Use € or for the euro symbol:

Example: The cost is 50€


4. Use ™ or for the trademark symbol:

Example: Bluedart™ provides delivery support.
